ASCOASCOM 'Dome Launcher' Dome Driver (VB)



This is the ASCOM Dome driver called 'Dome Launcher' by Michael J. Kidd <linuxkidd@gmail.com> http://linuxkidd.com

This Dome driver will perform an HTTP Get request against the configured URL for Dome OPEN and Dome CLOSE events.

 

Network Connectivity between your ASCOM running computer and the device which will be queried is required for this to operate.

 

Setup:

  1. Select 'Dome Launcher' from the ASCOM chooser for your Dome

  2. The first time you select 'Dome Launcher', you will be forced to choose 'Properties' and configure the URLs for the OPEN and CLOSE events.

  3. Now Now your ASCOM compliant software will generate these HTTP GET requests for the Dome OPEN and CLOSE events!


NOTE: The URLs must start with http:// or https://
